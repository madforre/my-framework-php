Router 클래스 설계

------------------------------------

라우터 모듈 만들고 사용한다.
Controller를 이용하여 View에 랜더링 할 수 있어야 한다. - 해결(get 방식만 가능)

------------------------------------

MVC 중 VC를 구현했으니 남은 Model을 구현한다.


HTTP 요청 -> 라우팅 -> ( 컨트롤러 <-> 모델(DB) ) -> 뷰에 랜더링

DB 커넥션과 DB 세팅을 모델에 설정하고
static으로 DB정보를 컨트롤러에 한번만 할당하고
나머지 컨트롤러에서는 상속하여 사용하자.
